# Patterns

Recurring flows assembled from components — onboarding, the daily mission
loop, the conversation/quiz loop, streaks & rewards, etc.

Nothing documented here yet. Note: screen-level **mocks** (a specific
product page wired up against `tokens/`) are deliberately kept *outside*
this repo — they're a consumer of the design system, not part of it. See
the `mockups/` folder alongside `wyso-design-system/` for those.

Add one doc per pattern once a flow has been designed end-to-end, rather
than as a single screen.
